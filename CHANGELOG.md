# Changelog

All notable changes to the Axis Twelve Admin Panel project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-06

### Added
- Initial release of Axis Twelve Admin Panel
- Dashboard page with interactive charts (Chart.js)
- Forms page with comprehensive form examples
- Typography page showcasing text styles
- Tables page with various table layouts
- Blank page template for custom content
- Top menu alternative layout
- Attribution page with credits and licenses
- Login page with split-screen layout
- Registration page with password strength indicator
- Custom color palette and theme system
- Responsive sidebar navigation
- Modular Axis Twelve CSS Framework integration
- MIT License

### Features
- **Dashboard (index.html)**
  - 4 stat cards with gradient backgrounds
  - Line chart for monthly sales
  - Bar chart for revenue comparison
  - Pie chart for traffic sources
  - Recent orders table
  - Responsive grid layout

- **Forms (forms.html)**
  - Text inputs, email, password fields
  - Select dropdowns and textareas
  - Checkboxes and radio buttons
  - File upload inputs
  - Form validation examples
  - Horizontal and vertical layouts

- **Typography (typography.html)**
  - Headings (h1-h6)
  - Paragraph styles
  - Text utilities (colors, alignment, transforms)
  - Lists (ordered, unordered, unstyled)
  - Code blocks and inline code
  - Blockquotes

- **Tables (tables.html)**
  - Basic table
  - Striped table
  - Bordered table
  - Hover effects
  - Action buttons
  - Responsive table wrapper

- **Authentication Pages**
  - Split-screen layout (branding left, form right)
  - Gradient background with animation
  - Form validation
  - Password strength indicator (register)
  - Responsive design (stacks on mobile)

### Technical Details
- Built with Axis Twelve CSS Framework v2.0
- Modular CSS architecture
- Chart.js v4.4.0 for data visualization
- Font Awesome 6.4.0 for icons
- Google Fonts (Inter)
- Vanilla JavaScript (no framework dependencies)
- HTML5 semantic markup
- CSS3 custom properties for theming

### Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
