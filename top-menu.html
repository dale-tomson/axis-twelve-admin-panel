Top Menu Layout

This is an alternative layout with horizontal navigation instead of the sidebar. The top menu provides easy access to all pages while maximizing the content area width.

Key features of this layout:

  • Full-width content area for better use of screen space
  • Sticky navigation bar that stays visible while scrolling
  • Horizontal menu items for quick navigation
  • Same dashboard widgets and functionality as the sidebar layout
  • Responsive design that adapts to different screen sizes

You can switch between this layout and the sidebar layout based on your preference.
